## Introduction

This is portfolio website template built with React, Typescript, Vite, and Tailwind CSS

## How to Start this project

### 1. Clone using git

```git
https://github.com/Bayusetiawan45/template-portfolio
```

### 2. Install dependencies

```bash
yarn install
```

### 3. Run development server

```bash
yarn dev
```

Open http://localhost:5173 with your browser to see the result.
