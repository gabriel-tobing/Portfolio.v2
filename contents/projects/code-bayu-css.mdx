## Introduction

This is open source CSS Library inspired by Bootstrap or Tailwind CSS. Elevate your web design with codebayucss, a versatile and innovative framework inspired by Bootstrap, empowering you to craft responsive and visually captivating websites with unparalleled flexibility.

## Features

You can use all css utilities from Code Bayu CSS like Bootstrap or Tailwind. Check full documentation [here](https://bayu-setiawan.gitbook.io/by.ui/getting-started/quickstart)

## NPM

This awesome package already published on npmjs. Check [here](https://www.npmjs.com/package/codebayucss)

## Package Installation

Install codebayucss as a dependency using NPM by running the following command:

```bash
npm i codebayucss
```

In React JS, just import this command to index.js file

```jsx
import "codebayucss/css/codebayu.css";
```

And yeah, Happy coding!
