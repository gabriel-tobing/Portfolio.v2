## Introduction

Pomolite is webtools that implement [pomodoro technique](https://en.wikipedia.org/wiki/Pomodoro_Technique)

## Features

### ⏰ Pomodoro Timer

The technique uses a timer to break down work into intervals, traditionally 25 minutes in length, separated by short breaks.

### 📝 Task Management

Organize your tasks within the app, assigning them to specific Pomodoro sessions. Stay on top of your to-do list and watch your productivity soar.

### ⌛️ Real-time Progress Tracking

Gain insights into your work patterns with real-time progress tracking. Visualize your accomplishments and identify areas for improvement.

### 🛎 Notification

Pomolite will notif you for each session end

### 🛡 Authentication

Login with google or github to secure your progress

## How to Start this project

### 1. Clone using git

```git
https://github.com/Bayusetiawan45/pomolite.git
```

### 2. Install dependencies

```bash
yarn install
```

### 3. Run development server

```bash
yarn dev
```

Open http://localhost:3000 with your browser to see the result.
