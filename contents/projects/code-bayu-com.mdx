## How to Start this project

### 1. Clone using git

```git
git clone https://github.com/Bayusetiawan45/v2.codebayu.com.git
```

### 2. Install dependencies

```bash
yarn install
```

### 3. Run development server

```bash
yarn dev
```

Open http://localhost:3000 with your browser to see the result.
